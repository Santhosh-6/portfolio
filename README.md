# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website developed to showcase my skills, projects, achievements, and technical expertise. This portfolio acts as my professional online presence where visitors can learn more about me and explore my work.

---

## 📌 About The Project

This portfolio website was created to present my profile, technical skills, projects, and contact information in a clean and interactive way.

The website includes:

- Personal introduction
- Technical skills
- Featured projects
- Resume section
- Contact section
- Responsive design for all devices

---

## 🚀 Features

✔ Responsive design for desktop, tablet, and mobile devices  
✔ Modern user interface  
✔ About section  
✔ Skills showcase  
✔ Project gallery  
✔ Resume download option  
✔ Contact form  
✔ Smooth scrolling navigation  
✔ Interactive animations

## ⚙ Installation

Clone repository:

```bash
git clone https://github.com/Santhosh-6/portfolio.git
```

Move to project directory:

```bash
cd portfolio
```

Open project:

```bash
index.html
```

Or use VS Code Live Server.

---

## ▶ Running the Project

Method 1:

Open:

```bash
index.html
```

Method 2:

Using VS Code:

1. Install Live Server extension
2. Right-click on index.html
3. Click "Open with Live Server"
